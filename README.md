Product Data Explorer

Backend service for scraping product data from **World of Books** using **NestJS**.

Project Overview
Product Data Explorer is a backend application built with NestJS that demonstrates:
- Web scraping architecture
- Modular backend structure
- Prisma schema usage
- REST API readiness

This project was developed as part of a backend assignment.

Tech Stack
- Node.js
- NestJS
- TypeScript
- Prisma ORM
